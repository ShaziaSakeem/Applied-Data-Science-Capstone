# IBM Applied Data Science Capstone

## Project Overview
This project is part of the IBM Applied Data Science Capstone course. The objective is to analyze SpaceX Falcon 9 launch data and build a machine learning model to predict first-stage landing success.

## Project Workflow
- Data Collection using SpaceX REST API
- Data Collection using Web Scraping
- Data Wrangling
- Exploratory Data Analysis (EDA) with Visualization
- Exploratory Data Analysis (EDA) with SQL
- Interactive Visual Analytics with Folium
- Interactive Dashboard with Plotly Dash
- Predictive Analysis using Machine Learning
